# School Vocab 3000 📚

현재 버전: **v1.0.0**

**School Vocab 3000**은 초중고 영어 기본 어휘 3,000개를 카드 한 장씩 학습하는 웹앱입니다.

단어, 국제음성기호, 한국어 발음, 품사, 뜻, 영어 예문과 한국어 번역을 한 화면에서 확인할 수 있습니다. 설치 없이 브라우저에서 실행할 수 있으며, 모바일 화면·음성 읽기·학습 상태 기록·자동 저장·저장 코드와 QR 공유를 지원합니다.

---

## 🎯 왜 만들었나

단어장을 공부하다 보면 단어를 읽는 것보다 **이미 아는 단어와 아직 익숙하지 않은 단어를 구분하고, 필요한 단어만 다시 보는 과정**이 더 번거로울 때가 있습니다.

- 3,000개 단어를 긴 목록이 아니라 한 장씩 집중해서 보고 싶다.
- 단어의 뜻뿐 아니라 발음과 실제 예문을 함께 확인하고 싶다.
- 낯선 단어와 익숙한 단어를 직접 분류하고 다시 골라 보고 싶다.
- 브라우저나 기기가 바뀌어도 학습 상태를 간단히 옮기고 싶다.

이 프로젝트는 이러한 학습 흐름을 복잡한 회원 가입이나 서버 없이, 하나의 가벼운 웹페이지 안에서 해결하는 데서 출발했습니다.

### 해결하고 싶었던 문제

1. 3,000개 단어를 카드 한 장씩 부담 없이 학습한다.
2. 단어·발음·뜻·예문을 한 화면에서 확인한다.
3. 학습 상태를 `낯섦 · 익힘 · 익숙`으로 나누어 필요한 단어만 복습한다.
4. 학습 기록을 브라우저에 자동 저장하고, 코드나 QR로 다른 기기에 옮긴다.

## 🧠 핵심 기능

| 기능 | 설명 |
|---|---|
| **단어 카드** | 영어 단어, IPA, 한국어 발음, 품사, 뜻, 예문과 번역을 한 장에 표시합니다. |
| **음성 읽기** | 단어와 영어 예문을 브라우저 음성으로 들을 수 있습니다. 보통·느림 속도를 지원합니다. |
| **순서대로 / 랜덤** | 원래 단어 순서와 무작위 학습 순서를 전환합니다. |
| **뜻 숨기기** | 뜻과 예문을 가린 뒤 스스로 떠올리고, 카드를 눌러 확인할 수 있습니다. |
| **단어 검색** | 철자를 입력해 현재 필터 안의 단어를 빠르게 찾습니다. |
| **난이도 필터** | `★ · ★★ · ☆` 조건을 조합해 원하는 범위만 표시합니다. 모든 항목을 끌 수도 있습니다. |
| **학습 상태** | 각 단어를 `낯섦 · 익힘 · 익숙`으로 기록하고 상태별로 모아 봅니다. |
| **미평가 보기** | 학습 상태 필터를 모두 끄면 아직 상태를 지정하지 않은 단어만 표시합니다. |
| **진행 번호** | 현재 학습 목록의 위치와 원본 단어 번호를 함께 표시합니다. |
| **자동 저장** | 설정, 현재 위치와 학습 상태를 브라우저에 저장해 다시 열어도 이어서 학습합니다. |
| **코드 저장·불러오기** | 학습 상태를 짧은 텍스트 코드로 저장하고 다른 브라우저에서 복원합니다. |
| **QR 공유** | 학습 상태가 담긴 QR을 웹앱 안에서 직접 생성합니다. |
| **모바일 지원** | 작은 화면에서도 카드와 필터를 사용할 수 있도록 반응형으로 구성했습니다. |
| **키보드 조작** | 방향키로 이동하고 숫자 키로 학습 상태를 빠르게 지정할 수 있습니다. |

## 🗂️ 카드에 담긴 정보

각 카드는 다음 정보를 포함합니다.

| 항목 | 설명 |
|---|---|
| **단어** | 기본 표제어와 필요한 경우 관련 형태를 함께 표시합니다. |
| **IPA** | 국제음성기호 발음을 표시합니다. |
| **한국어 발음** | 강세를 굵게 표시한 한국어식 발음 안내입니다. |
| **품사** | 명사, 동사, 형용사 등 품사를 표시합니다. |
| **뜻** | 주요 의미를 번호로 나누어 표시합니다. |
| **예문** | 각 의미를 확인할 수 있는 영어 예문과 한국어 번역을 제공합니다. |

단어와 예문은 각각 음성으로 들을 수 있습니다. 브라우저가 음성 합성을 지원하지 않으면 관련 버튼은 자동으로 숨겨집니다.

## 📈 학습 상태

단어마다 다음 세 상태 중 하나를 지정할 수 있습니다.

- **낯섦**: 아직 뜻이나 용법이 익숙하지 않은 단어
- **익힘**: 알고 있지만 더 확인하거나 반복할 필요가 있는 단어
- **익숙**: 뜻과 용법이 충분히 익숙한 단어

아무 상태도 지정하지 않은 단어는 **미평가**로 처리됩니다.

### 필터 동작

- **전체**: 평가 여부와 관계없이 현재 난이도 조건의 모든 단어를 표시합니다.
- **낯섦 · 익힘 · 익숙**: 하나 이상을 선택해 해당 상태만 모아 봅니다.
- **선택 없음**: 미평가 단어만 표시합니다.

상태 버튼을 다시 누르면 해당 평가가 해제되어 미평가로 돌아갑니다.

## 💾 자동 저장과 학습 상태 이동

### 브라우저 자동 저장

학습 상태와 앱 설정은 현재 브라우저의 `localStorage`에 저장됩니다.

- 페이지를 닫았다가 다시 열어도 이어서 학습할 수 있습니다.
- 서버 계정이나 로그인은 사용하지 않습니다.
- 다른 브라우저나 다른 기기의 기록과 자동으로 동기화되지는 않습니다.
- 브라우저 데이터를 삭제하면 저장된 기록도 함께 사라질 수 있습니다.

### 저장 코드

학습 상태 메뉴의 **저장**을 누르면 `B3K1`로 시작하는 코드가 생성됩니다.

```text
B3K1S:...
```

이 코드는 3,000개 단어의 `미평가 · 낯섦 · 익힘 · 익숙` 상태를 압축한 학습 기록입니다.

- **복사**: 코드를 클립보드에 복사합니다.
- **불러오기**: 다른 기기에서 복사한 코드를 붙여 넣어 학습 상태를 복원합니다.
- **초기화**: 모든 단어의 학습 상태를 미평가로 되돌립니다.

불러오기는 기존 기록에 덧붙이지 않고, 저장 코드에 담긴 상태로 전체 복원합니다. 난이도 필터, 순서, 음성 속도와 뜻 숨김 설정은 현재 기기의 값을 유지합니다.

## 📱 QR 공유

저장 팝업의 **QR**을 누르면 외부 QR 서비스 없이 웹앱 자체에서 QR 코드를 생성합니다.

### GitHub Pages에서 실행할 때

QR에는 현재 웹페이지 주소와 학습 상태 코드가 함께 들어갑니다.

```text
https://사용자명.github.io/school-vocab-3000/#s=B3K1S:...
```

다른 기기의 카메라로 QR을 스캔하고 링크를 열면 다음 순서로 동작합니다.

1. School Vocab 3000 페이지가 열립니다.
2. 주소에 포함된 학습 상태 코드를 검증합니다.
3. 학습 상태를 자동으로 불러옵니다.
4. 처리가 끝나면 주소에서 긴 상태 코드를 제거합니다.

### 다운로드한 `index.html`을 직접 열 때

로컬 파일은 다른 기기에서 접근할 수 있는 웹주소가 아니므로, QR에는 학습 상태 코드 자체가 들어갑니다.

- QR 생성과 스캔을 바로 시험할 수 있습니다.
- 카메라는 코드를 텍스트로 인식합니다.
- 웹페이지 자동 이동은 GitHub Pages처럼 외부에서 접속 가능한 주소에서 동작합니다.

학습 기록이 많을수록 QR이 촘촘해질 수 있습니다. 이 경우 화면 밝기를 높이고 QR을 정면에서 스캔하면 인식에 도움이 됩니다.

## ⚙️ 어떻게 동작하나 — 비개발자 설명

### 단일 HTML 웹앱

단어 데이터, 화면 디자인, 학습 기능과 QR 생성 코드가 모두 `index.html` 하나에 들어 있습니다.

별도의 설치 프로그램이나 데이터베이스 없이 브라우저에서 실행되며, 파일을 내려받아 직접 열면 인터넷 연결 없이도 기본 학습 기능을 사용할 수 있습니다.

### 필터와 학습 목록

난이도와 학습 상태 조건을 바꾸면 3,000개 전체 데이터에서 조건에 맞는 카드 목록을 다시 만듭니다.

랜덤 모드는 단어 데이터 자체를 바꾸지 않고, 현재 목록을 보는 순서만 섞습니다. 그래서 화면에는 다음 두 번호가 함께 표시됩니다.

```text
1 / 3000 · #1847
```

- `1 / 3000`: 현재 학습 목록에서의 진행 위치
- `#1847`: 원본 단어장의 고유 번호

### 학습 상태 저장 코드

단어마다 가능한 상태가 네 가지뿐이므로, 긴 JSON 파일 대신 상태를 압축한 문자열로 저장합니다.

코드에는 형식 버전과 검증값이 포함되어 있어, 일부가 잘리거나 잘못 붙여진 코드는 불러오지 않습니다.

### 자체 QR 생성

QR 이미지를 외부 사이트에서 받아오지 않고 브라우저 안에서 직접 계산해 SVG로 그립니다.

따라서 학습 상태를 QR 생성 서비스로 전송하지 않으며, 단일 HTML 구조도 유지합니다.

## 🕹️ 조작 방법

### 모바일 / 태블릿

- 화면 아래의 이전·다음 버튼으로 카드를 이동합니다.
- 단어나 예문의 스피커 버튼을 눌러 발음을 듣습니다.
- 뜻 숨김 상태에서는 카드를 눌러 뜻과 예문을 확인합니다.
- 필터의 꺾쇠 버튼을 눌러 난이도와 학습 상태를 설정합니다.

### PC

- 마우스로 버튼과 필터를 조작합니다.
- 단어 검색창에서 철자를 입력한 뒤 검색 결과를 선택합니다.
- 다음 키보드 단축키를 사용할 수 있습니다.

| 키 | 동작 |
|---|---|
| `←` | 이전 카드 |
| `→` | 다음 카드 |
| `Space` | 숨긴 뜻과 예문 보기 |
| `1` | 낯섦 지정 또는 해제 |
| `2` | 익힘 지정 또는 해제 |
| `3` | 익숙 지정 또는 해제 |

입력창이나 버튼에 포커스가 있을 때는 전역 단축키가 실행되지 않습니다.

## 🚀 실행과 배포

### 로컬에서 실행

1. `index.html`을 내려받습니다.
2. Chrome, Edge, Safari 등 브라우저로 파일을 엽니다.
3. 바로 단어 학습을 시작합니다.

### GitHub Pages로 배포

1. GitHub 저장소의 최상위 경로에 `index.html`과 `README.md`를 올립니다.
2. 저장소의 **Settings → Pages**로 이동합니다.
3. **Deploy from a branch**를 선택합니다.
4. `main` 브랜치와 `/(root)`를 선택해 저장합니다.
5. 생성된 Pages 주소로 접속합니다.

GitHub Pages에서 실행하면 학습 상태 QR을 스캔해 다른 기기에서 웹페이지를 열고 자동으로 복원할 수 있습니다.
