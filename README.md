# School Vocab 3000

초중고 필수 영단어 3,000개를 카드로 학습하는 오프라인 웹 앱입니다.

## 주요 기능

- 초중고 영어 기본 어휘 3,000개
- 단어 및 예문 음성 읽기
- 난이도별 필터
- 낯섦 · 익힘 · 익숙 학습 상태 기록
- 학습 상태 코드 저장 및 불러오기
- 순차 및 무작위 학습
- 뜻 숨기기
- 단어 검색
- 모바일 화면 지원
- 오프라인 단일 HTML 실행

## 실행 방법

웹사이트에 접속하거나 `index.html` 파일을 브라우저에서 열면 됩니다.

학습 상태는 현재 브라우저에 저장됩니다. 다른 기기로 옮길 때는  
`학습 상태 → 저장`에서 코드를 복사한 뒤, 다른 기기에서 불러오면 됩니다.
